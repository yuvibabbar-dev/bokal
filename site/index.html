Your cookies never leave your device

Trust isn't a feature bolted on — it's the whole design.

There is no Bokal server and no account. Bokal reads and writes cookies locally through Chrome's API and transmits nothing. The only time it contacts a network service is the optional Pro license check (via ExtensionPay and Stripe) — and only if you open the upgrade page. Free users are never contacted.

Bokal asks for no tabs permission and no site access at install; it requests access to the specific site you choose, only when you choose it. Read the full privacy policy →

Bokal's use of information received from Chrome APIs and any user data adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.
